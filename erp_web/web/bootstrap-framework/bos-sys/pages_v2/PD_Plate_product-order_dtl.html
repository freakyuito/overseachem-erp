质量要求
(1) 厚度 ±0.10mm  长度 +5mm  宽度 +5mm；
(2) 普通贴膜 保护膜有效宽度根据板材宽度，两端各3cm，保护膜位置居中；
(3)板面伍划痕，无杂物；
(4)表面压纹要清晰，纹路深度10丝以上；
(5)厚度均匀，覆膜一定要牢，板膜不允许有剥离现象，板面无水波纹，无发花现象；
(6)产品无明显色差，包装要好，记录字迹要清晰，装箱单总计一定准备无误；
(7)板材质量：需符合出厂检验标准，未达到的不允许出厂；
(8)薄膜厚度：20丝，确保纹路符合客户要求；
(9)颜色根据色号或来样生产，不能有色差；
(10)耐温提高5℃，模量增加10%，必须通过我司测试：23~25KGs摔箱测试不破裂， 冷冻零下15℃摔箱测试不破裂等符合国家箱包标准。
